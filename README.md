# LeetCode Solutions 🚀

This repository contains clean, well-structured solutions to various **LeetCode problems**.  
Each solution file is written so that you can **directly copy the code and paste it into the LeetCode code editor** to run it without any modifications.

---

## 📂 How to Use This Repository

1. Open any problem folder or file.
2. Choose the solution in your preferred programming language.
3. **Copy the entire code.**
4. **Paste it directly into the LeetCode online compiler/editor.**
5. Run ✔️ and submit 🚀.

All solutions follow LeetCode's required class/function structure so they work immediately.
